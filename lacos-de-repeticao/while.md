# While

O `while`loop percorre a interação enquanto a condição  for verdadeira.

```javascript
while (condição) {
  rotina
}
```

Exemplo:

```javascript
let passo = 1; 
while (passo <= 3) {
    console.log("Dei " + passo + " passos");
    passo++
}
console.log("Chegamos!!!");
```
